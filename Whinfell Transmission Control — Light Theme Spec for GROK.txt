## 1. Core tokens (light theme only)

Add these CSS custom properties under the `data-theme="light"` scope in `css/console_ia.css` (or equivalent theme root).[file:32][file:33]  

### 1.1 Surface and text colors

```css
/* Light mode root — inside [data-theme="light"] */
:root[data-theme="light"] {
  /* Surfaces */
  --wf-bg-page:    #F9FAFB;  /* off-white page, PPT slide-like */
  --wf-bg-card:    #FFFFFF;  /* cards / panels */
  --wf-bg-subtle:  #F3F4F6;  /* table banding, subtle blocks */

  /* Text */
  --wf-text-primary:   #1A1A1A;  /* deep neutral, PPT body text */
  --wf-text-secondary: #555555;  /* descriptions and subheaders */
  --wf-text-muted:     #808080;  /* hints, microcopy */
  --wf-text-inverse:   #FFFFFF;  /* text on dark chips/buttons */
}
```

Use these mappings:

- Console title, widget titles, main labels → `--wf-text-primary`.[page:1][file:33]  
- Descriptions, explanatory copy → `--wf-text-secondary`.  
- Keyboard hints and low‑priority status text → `--wf-text-muted`.
